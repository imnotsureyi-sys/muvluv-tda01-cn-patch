# 共通规则

## 原文来源

1. 只以日文原文字幕 / 日文 speaker 原文为依据。
2. 不使用英文文本进行翻译判断。
3. 不恢复英文兜底写回。
4. 不让空 JP 行自动触发模糊回写。
5. 如果 CSV 的 JP 为空，但游戏里显示了文本，优先重新抽取 / 对照 egpack，不能直接清空中文。

## 字幕和说话人

1. 正常对白保留中文对白引号。
2. 非对白说明文字不要强行加外层对白引号。
3. 日文纯停顿 / 纯标点时，中文也应保持停顿 / 标点，不额外加“嗯/啊/唔”。
4. 说话人表必须用日文 speaker 原文对中文名，不许用英文 speaker 名作为依据。
5. 多人说话人分隔符使用 `·`，不要用 `・`，不要用 `&`。

## 已确认术语方向

- `ボクサー`：`拳师号`
- `イーグル`：`鹰`
- `フェザント`：`雉鸡`
- `フェザント中隊`：`雉鸡中队`
- `サラマンダー`：`蝾螈`
- `クレイン`：`鹤`
- `ウォードッグ / Wardog / Wardogs`：`战犬`
- `surface pilot` 相关：`卫士`
- `pilot badge`：`卫士徽章`
- `大尉`：按用户要求译为 `上尉`
- `ユズちゃん`：`柚酱`
- `可憐 / 可憐さん`：人名，按语境译为 `可怜`，不要写成 `可怜女士/可怜小姐`
- `ＰＸ / PX`：保留 `PX`

## 日文原文里的拉丁字母

日文原文如果本来就是英文、拉丁字母、缩写或单位，中文也保留，不要擅自意译。

已确认保留对象包括：

`PX`、`OK`、`HQ`、`CP`、`IFF`、`FCS`、`DIA`、`D.C.`、`BC`、`AH`、`Need to know`、`JFK HIVE`、`OP`、`120mm`、`36mm`、`3cm`、`300km`

## 发布说明原则

- 明确标注非官方、非商业、测试版。
- 明确标注不含游戏本体、不提供破解、不改 exe、不碰存档。
- 明确标注制作者不懂日语，未完整日中人工校对。
- 可说明可能仍有错译、错字、空字幕、说话人错位、`Text ID Not Found` 类问题，欢迎截图反馈。
- 致谢主任保护协会提供 AGES 引擎汉化思路。
