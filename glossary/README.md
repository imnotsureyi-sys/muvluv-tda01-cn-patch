# 共通术语表说明

main 总库维护 Muv-Luv 世界观共通术语。

当前正式术语表仍保存在：

```text
outputs/glossary/muvluv_lunatranslator_full_glossary.csv
outputs/glossary/muvluv_lunatranslator_proper_nouns.tsv
outputs/glossary/muvluv_world_glossary_sources.md
```

## 使用规则

章节对话发现新术语时：

1. 先在本章记录候选。
2. 用户确认译法。
3. 同步到 main 的正式术语表。
4. 其他章节需要时再拉取 main 术语表。

章节对话不要直接改其他章节正文来统一术语。

## 术语优先级

1. 用户已确认译法
2. Muv-Luv 世界观通用译法
3. JP 原文语境
4. 音译/意译判断

英文槽不作为译名依据，只能在 JP 原文本身包含英文/拉丁字母时保留。
