# TDA00 Context Pack

This folder is a translation review aid for TDA00.

Hard rule: JP source text in `chapters/TDA00/baseline/tda00_jp_text_by_call_order.csv` is the only primary translation source. These notes are only for consistency checks, tone, terminology, and spotting likely model mistakes. If any note here conflicts with the JP line, follow the JP line.

Do not use English slots, old Chinese slots, fuzzy matches, or fallback text to overwrite the JP meaning.
