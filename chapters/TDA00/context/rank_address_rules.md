## Address Style

- Preserve rank/address when present in JP.
- Do not add speaker names to silent lines or short replies.
- If JP is only `「…………」`, final CN should normally remain `「…………」`.
- If JP says `は！`, military reply is usually `「是！」`, not a speaker name.
- If JP uses `了解`, prefer `了解/收到` depending on radio context; keep concise.

## Callsigns

- `ナイヴス` -> `黑刃`
- `ナイヴス１/２/３` -> `黑刃1/2/3`
- `オール・ナイヴス` -> `黑刃全机`
- `バスター１/２` -> `Buster 1/2` only if user later confirms; otherwise flag for user if a final choice is needed.
- `バスターズ` -> `Busters` only if user later confirms; otherwise flag.
