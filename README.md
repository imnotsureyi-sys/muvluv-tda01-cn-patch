## 安装方法

1. 关闭游戏。
2. 解压整个补丁文件夹。
3. 双击运行 `install.bat`。
4. 安装完成后，从 Steam 启动 TDA01。

卸载或恢复原状时，双击运行 `uninstall.bat`。

## 版本说明

- 当前版本是 Beta 0.1，仍在测试中，欢迎玩家带截图、章节、台词上下文反馈问题。
- 当前重建原则是以游戏日文字幕槽为准写入中文，不以英文文本为翻译依据。
- 制作者本人不懂日语，本版本没有经过完整的日中人工校对。
- TDA01 从中间部分开始已经用补丁实际游玩检查，并把发现的问题基本修复；前半部分仍欢迎玩家继续反馈。

## 已知风险

- 可能仍有错译、语病、术语不统一、繁体字/日文汉字残留。
- 可能仍有文本和语音、说话人名称、上下文顺序不匹配的问题。
- 极少数位置可能出现空字幕、乱码、缺字，或类似 Text ID Not Found 的缺失文本提示。
- 如果安装后仍看到旧文本，通常是旧缓存没有被当前补丁覆盖，建议关闭游戏后重新运行 `install.bat`。
