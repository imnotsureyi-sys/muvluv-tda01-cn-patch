# MuvLuv_TDA01_CN_Patch

Muv-Luv Unlimited: THE DAY AFTER TDA01 / episode:01 简体中文测试补丁。

这是非官方、非商业的同人简体中文补丁，仅供正版 Steam 玩家交流使用。本补丁不包含游戏本体，不提供破解，不修改 exe，不修改 Steam 原始游戏文件，不修改存档。

> 当前只公开发布 TDA01。TDA02 / TDA03 仍在整理和测试中，暂未公开发布。

## 翻译和校对说明

- 当前文本以游戏日文字幕槽为准进行中文写入，不以英文脚本作为翻译依据。
- 制作者本人不懂日语，本版本没有经过完整日中人工校对。
- TDA01 已进行一定实机游玩检查，但仍可能存在错译、语病、术语不统一、上下文不顺等问题。
- 欢迎玩家带截图、章节位置、台词上下文反馈问题。

## 安装方法

1. 先从 Steam 启动一次 TDA01，进入正文并看到第一句台词后退出游戏。
2. 下载并解压整个补丁文件夹。
3. 双击运行 `install.bat`。
4. 安装完成后，从 Steam 启动 TDA01 游玩。

## 卸载 / 恢复

双击运行 `uninstall.bat`，会恢复安装时备份的原文件。

## 注意事项

- 本补丁只适用于 TDA01 / episode:01。
- 请不要只复制 `payload` 文件夹，直接运行 `install.bat` 即可。
- 如果 Steam 验证游戏完整性或游戏更新后中文消失，请重新运行 `install.bat`。
- 本补丁会写入游戏本地缓存目录，并在安装前自动备份原缓存文件。
- 本补丁不修改 Steam 原始游戏文件和存档。

## 已知风险

- 可能仍有错译、语病、术语不一致、上下文不顺。
- 可能仍有说话人、语音、文本顺序不匹配的问题。
- 极少数位置仍可能出现空字幕、乱码、缺字，或类似 `Text ID Not Found` 的缺失文本提示。
