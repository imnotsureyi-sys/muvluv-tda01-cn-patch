# Muv-Luv Unlimited: THE DAY AFTER TDA00 简体中文补丁 beta0.1

这是 TDA00 / episode:00 的简体中文测试补丁。

## 版本定位

- 非官方、非商业同人补丁。
- 仅供正版 Steam 玩家测试交流。
- 不包含游戏本体，不提供破解，不修改 exe，不操作存档。
- 本章使用更完善的 JP baseline 工作流：先冻结日文原文槽、speaker、ruby、XML 调用顺序，再进行翻译与审计。

## 本版内容

- 剧本文本简体中文化。
- UI 图片使用中文化资源或官方英文资源兜底。
- 视频/开场演出字幕、章节相关文本已纳入汉化流程。
- 包含中文字体文件和一键安装脚本。

## 已做检查

- zip 完整性测试通过。
- JP baseline / 调用顺序 / speaker / ruby 对齐审计通过。
- 空中文、Text ID Not Found、异常重复、英文整句残留、术语问题：未发现硬性问题。
- 发布前已重新横向检查压缩包结构、中文槽、术语和显示槽风险。
- TDA00 已粗略过一遍，但没有像 TDA01-03 那样经过大量实机反馈。

## 已知风险

制作者不懂日语，当前版本未经过完整日中人工校对。可能仍有错译、语病、术语不一致、说话人错位、空字幕、缺字或 Text ID Not Found 类问题。
